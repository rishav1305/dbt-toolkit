# dbt-toolkit

Workflow-oriented dbt project management for Claude Code.

## Installation

```bash
claude plugin add dbt-toolkit
```

## Quick Start

```bash
/dbt-toolkit:dbt setup    # First-time configuration
/dbt-toolkit:dbt run      # Execute models
/dbt-toolkit:dbt test     # Run tests
/dbt-toolkit:dbt freshness # Check source freshness
/dbt-toolkit:dbt audit    # Health checks
/dbt-toolkit:dbt debug    # Troubleshoot issues
```
